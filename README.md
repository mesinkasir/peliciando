WORK :
+ create venv on pelican project
+ download source code pelicando
+ open venv\Lib\site-packages\pelican\themes
+ upload peliciando to themes folder
+ open pelicanconf.py
+ insert THEME = 'pelicianido' on pelicanconf.py
